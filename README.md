## About
Adaptive single page demo with the following stack implementation:
* HTML: BEM, a11y;
* CSS: Flexbox, Grid Layout;
* JavaScript, NPM, Webpack, Babel

Developed under the [Practicum](https://practicum.yandex.com/web/) online bootcamp.
